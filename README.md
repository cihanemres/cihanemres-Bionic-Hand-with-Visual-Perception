# Bionic Hand with Visual Perception

A cost-effective bionic hand system that replicates human hand movements using visual perception technology. This project combines Python, Arduino, and 3D printing to create a versatile tool for healthcare and safety applications.

## 📜 Project Overview
This project is designed to address challenges in remote operations in healthcare, safety, and hazardous environments. By mimicking hand movements in real-time, this system offers a low-cost and highly efficient solution.

---

## 🚀 Features
- Real-time hand tracking using OpenCV and MediaPipe.
- Seamless integration with Arduino and servo motors.
- Customizable calibration for user-specific hand movements.
- Cost-effective 3D-printed model.

---

## 🛠️ Components
- Arduino Uno
- Servo Motors (5 units)
- Camera
- PLA Filament for 3D Printing
- Connection Wires
- Python Libraries: OpenCV, MediaPipe, PyFirmata
